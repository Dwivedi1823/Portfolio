I'm a Developer

Experienced software developer with a robust background in PL/SQL database development, including designing, implementing, and optimizing database solutions. Driven to expand technical expertise, I have transitioned to full-stack web development, becoming proficient in front-end technologies (HTML, CSS, JavaScript, React.js) and back-end development (Node.js). About Me

My journey as a software developer started with a deep dive into PL/SQL development, where I gained a solid understanding of database systems. I'm a firm believer in continuous learning, so I embraced the challenge of full-stack development. I'm now comfortable working with technologies like HTML, CSS, JavaScript, React.js, and Node.js. What excites me most is connecting the dots between the database and the user interface to deliver smooth and effective applications.
